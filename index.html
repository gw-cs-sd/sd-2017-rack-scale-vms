Neel Shah: Rack Scale VMs

Black magic.
A softare solution for rack scale architecture. Rack scale architecture provides the opposite of traditional virtualizatoin--it enables one virtual machine to access the hardware of other physical machines in the same rack. This project leverages the Xen hypervisor, Bitdefender's libbdvmi, and novel memory management algorithms to recreate rack scale architecture with only software on commodity hardware.
